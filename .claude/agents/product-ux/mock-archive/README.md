# モックアーカイブ

`product-ux` **直下**にあった **過去版**の `mock-NNN.html` を置く場所です。

- **最新版だけ**は直下の `mock-NNN.html` に1本だけ置く。
- **新しいモックを作る前に**、直下のすべての `mock-*.html` をここへ **移動**する（削除しない）。
- 運用の詳細は `../agent.md` の「モック生成ルール」を参照。

過去にファイルが消えたのは、誤った一括巻き戻しで `mock-archive` 内の HTML まで削除されたためです。履歴は **このフォルダに退避**し、本番パスとは別に管理する。
