# 日次X投稿案 — 固定ルール（コピー用）

このファイルを `daily-proposals-rules.md` にコピーして編集してください（同じ `docs/` フォルダ内）。

## アカウント・目的

- （例）個人開発の進捗共有 / プロダクト告知 / 技術メモ など
- フォロワー想定: （例）エンジニア中心

## トーン・文体

- です・ます調 / タメ口 など、希望を書く
- 煽り・断定的な攻撃は避ける

## 投稿の型

- 案は **3パターン** 出す（A/B/C）
- 各案 **280文字以内**（必要ならスレ用に「続き」案も）
- ハッシュタグ: **付けない** / **2つまで** など

## 禁止・注意

- 未リリース機能を「出ました」と書かない
- 他者・他社を名指しで貶めない
- 個人情報・秘密情報を書かない

## 出力フォーマット（この見出し構造で返す）

1. `## 要約（今日の文脈）` … 1〜3行
2. `## 投稿案` … A/B/C
3. `## 任意: 英語版` … 必要なら
