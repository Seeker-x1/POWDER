# 日次X投稿案 — 固定ルール（運用で編集する）

<!-- docs/daily-proposals-rules.example.md に詳しい例あり -->

## アカウント・目的

- 開発の進捗・学びの共有

## トーン・文体

- です・ます調。読みやすく短く。

## 投稿の型

- 案は **3パターン**（A/B/C）
- 各案 **280文字以内**
- ハッシュタグは **任意（最大2つ）**

## 禁止・注意

- 未確定の事実を確定として書かない
- 個人情報・APIキー・社外秘を含めない

## 出力フォーマット

1. `## 要約（今日の文脈）` … 1〜3行
2. `## 投稿案` … A/B/C
3. `## 任意: 英語版` … 必要なら
