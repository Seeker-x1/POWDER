プライバシーポリシー（簡易）

最終更新: 2026-06-30

JapawSerch（POWDER）は個人開発の参考情報サイトです。

収集する情報

  • 本サイトはログイン機能を提供しません。
  • アクセス解析（Google Analytics 等）は現時点では組み込んでいません。
  • ブラウザの localStorage に、気象庁降雪タイルの最終表示時刻など、表示のための設定のみを保存することがあります（端末内のみ）。

外部サービス

天気・地図表示のため、利用者のブラウザから次の第三者へリクエストが送信されます。

  • Open-Meteo（予報）
  • 気象庁（積雪・タイル）
  • OpenStreetMap / CARTO（地図）

各サービスのプライバシーポリシーが適用されます。

お問い合わせ

GitHub リポジトリの Issue をご利用ください: momentum-create/POWDER

Privacy (summary)

No accounts, no analytics at this time. Optional localStorage for UI state on your device only. Your browser calls Open-Meteo, JMA, OSM, and CARTO directly. Contact via GitHub Issues.
